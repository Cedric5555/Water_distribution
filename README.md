# Water Distribution System

## Project Description

The Water Distribution System is an Object-Oriented Programming project developed in Python.

The purpose of this project is to model a simplified water management system where different water sources provide water, different consumers use water, and treatment systems improve water quality.

The project demonstrates important OOP concepts such as:

- Abstraction
- Encapsulation
- Inheritance
- Polymorphism
- Composition
- Separation of responsibilities


## Project Objectives

The system allows users to:

- Manage different types of water sources
- Manage different types of water consumers
- Apply different water treatment methods
- Calculate water production and consumption
- Generate reports about the current water network


## System Components

### Water Sources

The system contains different water sources:

- River
- Well
- Reservoir

Each source has its own way of producing water.


### Consumers

The system contains different water consumers:

- House
- Farm
- Factory

Each consumer has different water requirements.


### Water Treatment

The system supports different treatment methods:

- Sand Filter
- UV Treatment

Each treatment method has its own purification process and cost.


## Object-Oriented Design

The project uses abstract classes to define common behaviour.

Examples:

- WaterSource is the base class for all water sources
- Consumer is the base class for all consumers
- WaterTreatment is the base class for all treatment methods


The system uses polymorphism so that different objects can be handled through their common interfaces.
